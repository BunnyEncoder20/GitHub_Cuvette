_MarkDown Syntaxes_
bold : **text** or __text__
italic : *text* or _text_
bold_italic : ***text***
Strikethrough : ~~text~~
subscript : <sub>text</sub>
superscript : <sup></sup>

Quote a text : >text
Quoting Code : ``` code ```
Inserting Links : [GitHub Pages](https://pages.github.com/)

Bullets and Lists : 
Unordered :  -, *, or +
- one
* sub
+ three

Ordered : {any number} Item : 
1. one 
2. two
3. Three

Sub point {tab} - {space}
1. Point
    - sub point
