Position Properties

  1. Static : it's like a default thing, doesn't change anything
  2. Absolute : positions itself according to it's FIRST POSITION PARENT element (Position Parent = element with position property)
  3. Relative : Moves that much px according to it's original position (the original position of the element without any position property)
  4. Fixed : It's a form of Absolute property. It stays fiixed on the screen (irresponsive to scrolling also)
  5. Sticky : it's kinda weird it is fixed until we scroll then
