Transition proerties

  • Transition-delay : The amount of time given before an transition starts
  • Transition-duration : The amount time from start to end state of transition
  • Transition-property : which css properties you want to transition
  • Transition-timing-function : at wot kinda timing you want (eg : fast in beginning, slow at end)
  • Transition : short hand for specifying transition-property & transition-duration


Animation proerties

  • animation-name : specifies the name of that animation
  • keyframes : used to define the keyframes (points of time / frames) of the animation
  • animation-iteration-count : specifies how many times the animation is supposed to occur
